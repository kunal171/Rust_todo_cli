
# TO DO LIST

This is a project on Rust to create a "***TO_DO_List** with mutilple User iterface*".


#### **Project contains**

- **User loging page**
    - "*Sign Up*" option for new User
    - "*Sign in*" option for the existing user.

- **To_Do list interface**
    - *Create list*
    - *Update List*
        - *Add Task*
        - *Remove Task*
        - *Mark As Done*
    - *Delete List*
